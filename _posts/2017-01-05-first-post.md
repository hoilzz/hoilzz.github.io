## 작성 방법

Jekyll은 Post 상단의 `front matter`라는 양식을 작성해야한다. 현재 작성된 포스트의 `front matter`는 다음과 같다.

*front matter*

```
---
layout: post
title: 첫 포스트
description: "이것은 나의 첫번째 포스트다."
tags:
  - sample post
  - link post
comments: true
share: true
date: 2016-01-05
---
```
