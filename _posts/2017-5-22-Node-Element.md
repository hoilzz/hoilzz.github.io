# Node와 Element 차이

Node는 DOM계층에서 가장 기본이 되는 데이터 타입이다.

Node를 구현한 여러 오브젝트가 있는데 노드 타입으로 구분 가능하다.
`elementNode`, `textNode`, `attributeNode`, `commentNode` 등이 있다.

Element는 시작태그에서 종료 태그를 가진 모든 것이다.


## Summary

Node는 엘리먼트의 상위 개념이다.
